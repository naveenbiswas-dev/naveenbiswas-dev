## 🚀 Featured Projects

### 🔐 RBAC Admin Dashboard
> Role-Based Access Control system with per-role module visibility — mirrors real enterprise auth patterns

**Stack:** React.js · TypeScript · Node.js · Express.js · JWT · Bootstrap 5

- Granular permission system: Admin, Manager, Viewer roles with route guards
- Backend middleware enforcing role checks on every protected API endpoint
- Dynamic sidebar that renders only permitted modules per user role
- Clean JWT auth flow with refresh token handling

---

### 📋 Dynamic Survey Form Builder
> Conditional-logic form engine with real-time validation — based on production survey work for US clients

**Stack:** React.js · TypeScript · Node.js · Express.js · REST APIs

- Multi-step forms with conditional branching (show/hide fields based on answers)
- Real-time inline validation with contextual error messages
- Form state persisted to backend via RESTful APIs
- Progress indicator to reduce survey drop-off


---

### 📊 Data Table Dashboard
> Filterable, paginated data dashboard — the kind of table found in every real SaaS product

**Stack:** React.js · JavaScript (ES6+) · REST APIs · Axios · CSS3

- Server-side filtering, sorting, and pagination
- Column-level search with debounced API calls
- Responsive layout for mobile and desktop
- Clean component architecture — table, filters, and pagination fully decoupled
